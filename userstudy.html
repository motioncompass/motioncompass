Environment

Setup for network monitoring:

  • Amazon Blink XT Camera

  • Netgear R6300 V2 Wireless AP (operating in 2.4 GHz mode)

  • Cox network - Oklahoma

  • Rooted Motorola Nexus 6

  • Kali Nethunter ROM 2021.1

  • Aircrack-ng 1.6

Experiment

We recruited 5 volunteers and asked each of them to perform MotionCompass to figure out the location of a hidden wireless camera randomly selected and deployed in the aforementioned outdoor or indoor environment.

Result - localization time

We output the localization time into csv format (txt) files and then use Python to do data processing. Here are the codes we used to generate the final result:

For User 1:

import  csv
from  statistics  import  mean
with  open('1.txt', newline='') as  f:
reader = csv.reader(f)
data = list(reader)
for  i  in  range(0, len(data[0])):
data[0][i] = int(data[0][i])
print("Min time of user 1 is ", min(data[0]))
print("Max time of user 1 is ", max(data[0]))
print("Average time of user 1 is ", str(mean(data[0])))

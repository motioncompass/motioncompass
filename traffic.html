Environment

Setup for network monitoring:

  • X86 router - Openwrt Koolshare mod V2.33 r12074-007caa48d1

  • netdata.cloud service built-in to monitor wireless traffic

  • Cox network - Oklahoma

  • Netgear R6300 V2 wireless AP (operating in 2.4 GHz mode)

  • Arlo Pro Base Station (hub) connected with the router using Ethernet port 1

Devices used in the network:

  • Device 1: An Arlo Pro 2 wireless camera connected with the hub - 2.4 GHz WiFi

  • Device 2: iPhone SE (2nd Generation) - 2.4 GHz WiFi

  • Device 3: Surface pro 2 - 2.4 GHz WiFi

The Netgear R6300 V2 wireless AP is connected to the X86 router, and all wireless traffic flows are collected on X86 router. We log the inbound and outbound packets and then process the data with Matlab.

We observe the data streams within 50 seconds, during which the camera has been triggered while the other two devices are in normal use mode.

The following code is used to generate the figure:

LabelFontSize = 16
cam = plot(x,y_cam,'linewidth',2,'Color',[1 0 0]);
hold  on
o1 = plot(x,y_2,'linewidth',2,'LineStyle','--','Color',[0  0  0]);
hold  on
o2 = plot(x,y_3,'linewidth',2,'LineStyle',':','Color',[0  0  1]);
hold  off
legend([cam,o1,o2],'Camera traffic', 'Non-camera traffic 1','Non-camera traffic 2');
xlabel('Time (sec)','FontSize',LabelFontSize);
ylabel('Count of New Packets (pkts)','FontSize',LabelFontSize);
set(gca,'FontSize',LabelFontSize)
ylim([0 200]);
xlim([0 49]);
